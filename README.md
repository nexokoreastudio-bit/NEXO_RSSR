# 넥소 X 성공운 랜딩페이지

성공운 카페 공동구매용 **넥소 & 성공운** 전자칠판 랜딩 페이지입니다.

## 기술 스택

- **React 18** + **Vite**
- **Tailwind CSS**
- **lucide-react** (아이콘)

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (http://localhost:5173)
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

## 페이지 구성

1. **네비게이션** – 공구 포인트, 과목별 솔루션, 비용 절감, 안심 서비스 앵커
2. **히어로** – 한 줄 메시지 + CTA (한정수량)
3. **공구 포인트** – 일반 전자칠판 vs 넥소 X 성공운 비교
4. **과목별 솔루션** – 수학 쌍둥이 문제 추출 소개
5. **비용 절감** – 학생 수 슬라이더 + 월/연 절감액 계산기
6. **안심 서비스** – A/S, 원격 지원, 수업 노하우
7. **최종 CTA** – 상담 신청 폼 (성함, 연락처, 희망 과목)
8. **푸터** – 넥소, 저작권, 성공운 카페 링크

## 디자인 방향

- **타깃**: 성공운 카페 원장님 (학원 운영자)
- **톤**: 신뢰감 있는 인디고 계열 + 명확한 혜택 전달
- **반응형**: 모바일/태블릿/데스크톱 대응

이 코드를 기반으로 섹션 추가·수정·디자인 개선을 이어가면 됩니다.
